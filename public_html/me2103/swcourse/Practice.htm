1) Part Modeling for Cover

CREATING THE BASE FEATURE

1.1) Click on New and select . Select the Front plane and click on Sketch . (You have to select a plane before you can sketch.)


1.2) Select Rectangle. Move the pointer to the origin (to capture a coincident relation). Left-click and move the pointer to create a rectangle. Then, click on the Smart Dimension tool . Click on the breadth and type 150 mm. Click on the length and set to 300 mm. 


1.3) Click on Extruded Boss/Base . Select Blind and type Depth 100 mm. Click on to complete. (Since this is the first feature in this part modeling, this feature is also known as the base feature.)

 

1.4) Select the highlighted face as shown. Start a sketch by selecting Circle . Sketch a circle of diameter 100 mm and centered 75 mm from the top and left sides by using Smart Dimension tool to fix the distance.


1.5) Click on Extruded Boss/Base. Select type Blind and Depth 50 mm. (Ensure the option 'Merge result' is checked to produce a single solid.) Click on to complete.

 

CREATING A STEP
1.6) Select the highlighted face as shown. Sketch a Rectangle of width 130 mm. (The top, bottom and right edges have to be collinear/coincident with the corresponding edges of the rectangular feature as shown.)

1.7) Click on Extruded Cut. Select type Blind and Depth 30 mm. (Click on Reverse  if the cutting is not in the correct direction). Click on to complete.

 


CREATING A HEXAGONAL HOLE
1.8) Select the face as shown. Click on the Polygon icon . Specify the no. of sides to be 6 and sketch a hexagon on the face.

 


1.9) Specify the position of the hexagon using the Smart Dimension tool by giving a dimension of 75 mm from the base to the center of the hexagon. Similarly, specify the size of the hexagon by providing a diameter of 50 mm for the internal circle of the hexagon.


1.10) Give a distance of 40 mm from the vertex of the hexagon to the side.


1.11) Add a relation such that the side of the hexagon is parallel to the base. Under Display/Delete Relations , click on Add Relation . Select the bottom edge of the part and the base of the hexagon highlighted in green as shown. Select the option . Click on when done.

 

1.12) Next, click on Extruded Cut and select Through All. Click on  to complete.

 

CREATING A CIRCULAR HOLE THROUGH THE BOSS
1.13) Select the face of the boss. Sketch a Circle of diameter 50 mm such that it concentrics with the circular boss.


**Note: There are two methods of capturing a concentric relation.
(1) Move the pointer over the circular edge. The centre of the circle will appear as a black cross. Sketching a circle from that point would ensure that the circular sketch would be concentric to the part.
(2) Use Add Relationto define the relations between the sketched circle and circular edge such that they are concentric.
　

1.14) Click on Extruded Cut. Select Through All. Click on to complete.

 

FILLETING THE FOUR EDGES
1.15) Click on Fillet. Select the four edges as shown and change fillet radius to 20 mm. Click on to complete.

 

FILLETING THE FOUR FACES
1.16) Click on Fillet. Select the three faces as shown and change fillet radius to 5 mm. Click on  to complete.

SHELLING THE PART
1.17) Select the back face to begin. Click on Shell . Set thickness to 5 mm. (Select 'Show preview' to see the preview of the shell.) Click on  to complete.


 

SAVING THE PART
1.18) Click Save or click File -> Save.  Save the solid part as "Cover".

3) Assembly Modeling for Cover & Base

3.1) SIMPLE ASSEMBLY MODELING EXERCISE

Based on the basic knowledge on assembly modeling, an assembly could be created with 2 components created in Step 1 and Step 2. Typically, 3 mating relationships are needed to fully constrain the 6 degrees of freedom in an assembly.

3.1.1) Click on New and select .

3.1.2) Click on "Browse" and select the "Cover" solid part saved.

 

3.1.3) Left-click to place the component. The first component added into the assembly would be fixed.

 

INSERTING THE SECOND COMPONENT
3.1.4) Click on Insert Component and "Browse" for the part: Base.SLDPRT. Left click anywhere in the graphics area of the assembly to place the second component. (This second component is not fixed at this moment. It can be moved using Move Component and rotated using Rotate Component .)


　

ADDING THE FIRST MATING RELATION

3.1.5) Click on Mate   and select the pair of faces as shown. Select the Coincident  option. Select to accept the mate relation. (This aligns the two faces together. The second component can be moved freely using the Move Component  but could now rotate about one orientation only using Rotate Component .)


　

ADDING THE SECOND MATING RELATION
3.1.6) Select another pair of faces as shown. Select the Coincident option. Select    to accept the mate relation. (This aligns these two faces together. The second component can be moved in one direction using Move Component  but could not rotate using Rotate Component  .)


　

ADDING THE 3RD (AND LAST) MATING RELATION
3.1.7) Next, select the pair of edges as shown. Select the Coincident option. Select    to accept the mate relation. (This aligns these two edges together. The second component cannot be moved using Move Component and rotated using the Rotate Component  ). Click on  again. The components are now fully defined in the assembly and all degrees of freedom are constrained.


Note: This is just one suggested method of assembly modeling for this example. There are many other combinations of ways of constraining the parts.
　

SAVING THE ASSEMBLY
3.1.8) Click Save   or click File -> Save. Save this assembly as "Container_assembly".

Creating an eDrawing

To allow users without the SolidWorks software to view SolidWorks parts or assemblies, an eDrawing can be sent to them to allow them to view the model. eDrawings also allows the models to be animated. eDrawings can be published from SolidWorks part, assembly and drawing files. There are different file extensions to differentiate the different file types that were being published. eDrawings files that are saved without the built-in viewer requires a viewer to view the files. However, eDrawings files that are saved with the built-in viewer are standalone files, which can be run on any Windows-based PC without the need of any viewer.

The common file types that are used are:

SolidWorks file (file extension)

eDrawings file extension(without built-in viewer)

eDrawings file extension(with built-in viewer)

 

Requires eDrawings viewer

Do not require eDrawings viewer

Part (*.sldprt)

*.eprt

*.exe

Assembly (*.sldasm)

*.easm

*.exe

Drawing (*.slddrw)

*.edrw

*.exe

Below is a short exercise on how to create eDrawings.

Loading of eDrawings application
The eDrawings application has been integrated into SolidWorks and there is now no need to separately activate it.

a) eDrawings from SolidWorks part model (35_02a.swf) and (35_02b.swf)
1. Open the desired SolidWorks part model using SolidWorks.

There are two methods of creating an eDrawings file:

2. Within the SolidWorks application interface, select File -> Save As.
Select the file type that is to be saved as eDrawing (*.eprt). Then, select to save the eDrawings file.

3. Browse for the eDrawing file in the directory that you had saved and double-click to open the file. This file would be opened in the eDrawings viewer.

If you want to save the eDrawings file as a standalone self executable file, select File -> Save As.
Select the file type eDrawing Executable Files (*.exe). The name of this self-executable file would thus be Cover.exe.

4. The second method involves using the eDrawings toolbar or using File -> Publish to eDrawings from the SolidWorks interface.

b) eDrawings from SolidWorks assembly
The method of creating eDrawings from SolidWorks assembly is the same as creating eDrawings from SolidWorks part model.

1. Open the desired SolidWorks assembly model using SolidWorks.

2. Within the SolidWorks application interface, select File -> Save As.
Select the file type that is to be saved as eDrawing (*.easm). Then, select to save the eDrawings file.

3. Browse for the eDrawing file in the directory that you had saved and double-click to open the file in the eDrawings viewer.

4. The second method involves using the eDrawings toolbar or using File -> Publish to eDrawings from the SolidWorks interface.

c) eDrawings from SolidWorks drawing
The method of creating eDrawings from SolidWorks drawing is identical to the examples above.

1. Open the desired SolidWorks drawing using SolidWorks.

2. Within the SolidWorks application interface, select File -> Save As. 
Select the file type that is to be saved as eDrawing (*.edrw). Then, select to save the eDrawings file.

3. Browse for the eDrawing file in the directory that you had saved and double-click to open the file in the eDrawings viewer.

4. The second method involves using the eDrawings toolbar or using File -> Publish to eDrawings from the SolidWorks interface.

To view the model as shaded, depress the Shaded button. To animate the model, in the Drawing mode, click on animate and then play in the task bar.

 

d) eDrawings Password (35_05.swf)
The purpose of eDrawing password is for user to have an added protection to their eDrawings Professional files during export.

To add a password to an eDrawing file:
1. When saving a eDrawings file, select Password.

   

2. In the dialog box:
      a.   Select Password required to open document
      b.   Type a password.
      c.   Type the password again in Confirm password.
      d.   Click OK.
3. In the Save As dialog box, click Save to save the eDrawings file with password protection
