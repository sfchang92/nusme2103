Assembly:

2.1) To create a new assembly, open up solidworks software and click on New.
If you are using Solidworks without any given template, a box will appear as shown below. Select Assembly and click OK to create a new Assembly.



If you are using a computer at the PC clusters in school, ME2103 template is available. Click on Advanced on the bottom left corner and the ME2103 template will appear as shown below. To create a new assembly, select ME2103_Assem as indicated in the red box and click Ok.



2.2) To insert parts for assembly, click on "Browse" as indicated in the red box and search for the part file.



2.3) After selecting the part file, left-click on the assembly area to place the component.
The first component placed will be fixed (i.e. cannot be moved or rotated). This is denoted by the symbol "f" in the Feature Manager design tree as shown below.
Subsequent components added will be indicated by a symbol "-" and these components will not be constrained in any way.



2.4) Subsequent components can be added by selecting "Insert Components" as shown below.




2.5) The assembly can be zoomed in, zoomed out and rotated using the same method as mentioned under Modeling.

2.6) To move and rotate only one particular component, click on Move Component as shown below and select either Move Component or Rotate Component.
Move cursor to the particular component, left-click and hold to move or rotate the component.



2.7) To start mating the components, click on "Mate" and select the faces of the components to be mated.



2.8) There are a few mate relationships that can be used when mating two components:
• Coincident: Faces of the components would be in contact with each other
• Perpendicular: Components are constrained at 90° to each other (Aligned and anti-aligned do not apply here)
• Tangent: Applies to components with cylindrical faces. The cylindrical face will always be tangent to the selected surface of the other component.
• Concentric: Applies to components with cylindrical faces. Components share the same axis of rotation
• Parallel: Components are constrained such that they are aligned parallel to each other
• Distance: Components are constrained with a specified distance between them
• Angle: Components are constrained at a specified angle to each other

2.9) In addition to the mate relationships that are mentioned, there are also alignment conditions:
• Aligned: Normal or axis vectors of components for the selected faces point in the same direction
• Anti-Aligned (On): Normal or axis vectors of components for the selected faces point in opposite directions
• Closest: Components are either aligned or anti-aligned, depending on which condition can be satisfied with the least movement
