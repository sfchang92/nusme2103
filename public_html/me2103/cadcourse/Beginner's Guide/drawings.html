Drawings:

3.1) To create a new drawing, open up solidworks software and click on New.
If you are using Solidworks without any given template, a box will appear as shown below. Select Drawing and click OK to create a new drawing.



If you are using a computer at the PC clusters in school, ME2103 template is available. Click on Advanced on the bottom left corner and the ME2103 template will appear as shown below. To create a new drawing, select ME2103_Draw as indicated in the red box and click OK.



3.2) To insert the model, click on "Browse" and search for the file.



3.3) Next, select the orientation (indicated in the red box) of the model to be placed on the drawing sheet and the scale (indicated in the blue box) to be used for this drawing.



3.4) The first view can then be placed onto the drawing sheet:
Left-click on the drawing sheet. When done, click on the tick or simply press Esc.

3.5)This first view will be the "parent view" and the subsequent "child views" can be created by clicking on "Projected View" as shown.
Then move your cursor with respect to the position of the first view and left-click to create the desired "child views".



3.6) To insert an isometric view, click on "Model View" as shown in the red box.



Double-click the file browsed earlier and select Isometric under orientation. Left-click to place the isometric view.

 

3.7) To add Note:
Click on "Note" as shown below and left-click on the drawing sheet where you want to type.



3.8) To add Projection Symbol:
Click on "Note" and select "Add Symbol" as indicated in the red box below.



Under "Symbol library", select "Projection Symbol". Select "Third Angle Projection" and click OK.
Left-click on the drawing sheet to place the projection symbol.
