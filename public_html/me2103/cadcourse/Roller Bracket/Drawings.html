Roller Bracket

Drawings

 

Required:

1. Add a fully assembled sectional front elevation on the center line X-X of the base.

2. Add a plan view of the assembly.

3. Add an end elevation looking from the right of the front elevation.

4. Add the 5 dimensions indicated with *.

5. Indicate clearly the angle of projection that you are using.

7.1) To create a new drawing document: Click on New and select    to create a new drawing.

7.2) To import the roller bracket file:

 Under 'Part/Assembly to Insert', click on Browse and select the Roller Bracket Assembly saved in step 6.9.

7.3) To add a Model view:

Under 'Orientation', select Right . Under 'Display Style', select Hidden lines removed. Under 'Scale', select Use custom scale and set scale as 1:2. Position the first view as shown. Click on or Press Esc on keyboard to finish.

  

7.4) To add the Plan view: Click on Projected View , position the plan view as shown and click on the drawing sheet to place the view. Click on or press Esc on keyboard to finish.


7.5) Hide the first view: Right click on the first view and select Hide.

 

 

 

 

 

7.6) To create section view X-X: 
Click on the plan view to set it as the active view. Click on Section View . Select Horizontal cutting line and position the cutting plane such that it passes through the center as shown and click to place it on the view.

 

Click on and a section scope window will appear.
Check Auto Hatching. Select the Spindle and the Ribs from both brackets from the drop down list on the left as shown to exclude it from sectioning.  (Ensure that the cutting is in the correct direction.)

 

Click Ok and place the sectional view as shown.


 

Change the alphabet to X and click on .

 

 

 

 

7.7) Changing the line style: 

Select Options:

Click on Document Properties -> View Labels -> Section.
Under 'Line Style', select the last line style from the drop down list. Click Ok.



7.8) To add the end elevation view: Select the section view, click on Projected View and place the end elevation view to the right of the section view as shown.

7.9) To add the isometric view: Click on Model View and repeat the procedures in step 7.2 and 7.3 except under 'Orientation', select Isometric. Place the Isometric View as shown. Click .


 

 

 

 

7.10) Editing the Tangent edges: Right click on the Plan view, go to Tangent Edge and select Tangent Edges with Font. Do the same for the End elevation view and Isometric view. As for the sectional view X-X, select Tangent Edges Removed.
