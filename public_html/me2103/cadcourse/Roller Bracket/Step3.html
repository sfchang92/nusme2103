Roller Bracket

Step 3: Creating the Roller

3.1) To create a new document : Click on New and select   to create a new Part document.

3.2) To create the roller: Select Right plane in the Features Manager. (You have to select a plane before you can sketch).

a. By using Centerline and Line , sketch the following diagram as shown and use Smart Dimension  to set the appropriate dimensions. (Draw centerline horizontally and vertically across the origin).



b. Click on Revolved Boss/Base . Select the horizontal center-line as Axis of Revolution, select type Blind and set Angle as 360 degrees. Click on to finish.

   

3.3) Filleting the Roller: Click on Fillet . Select the 4 circumferences on both inner sides of the roller and set the fillet radius to 6mm. Click on to finish.

 

3.4)  You can now save this model by clicking File -> Save. Name this file as "Roller".
