    • Step 1: Creating the Bracket

    1.1) To create a new document : Click on New and select    to create a new Part document.

    1.2) To create the base : Select Right plane in the Features Manager. (You have to select a plane before you can sketch).

    a. Click on Rectangle  and sketch a rectangle of any size. Next, click on Smart Dimension , select the length of the rectangle and set the dimension to 42mm. Select the breadth and set the dimension to 10mm.

    b. Click on Circle and sketch a circle on the top right hand corner of the rectangle as shown. Click on Smart Dimension and set its diameter to 10mm. 


    c. Click on Trim Entitiesand select 'Trim to closest' to trim off the excess lines and curves to obtain the sketch as shown. 


    d. Click on Extruded Boss/Base , select type Blind and set depth at 84mm. Click on    to finish.

     

    1.3) To create the vertical structure :

    a. Click on Top view and select the top face as the sketching plane. Click on Rectangle  and sketch a rectangle touching the top edge of the plane as shown. Click on Smart Dimension , select the length of the rectangle and set the dimension to 32mm. Select the breadth and set the dimension to 9mm. Position the rectangle 26mm away from the left edge of the plane as shown.


     

    b. Click on Extruded Boss/Base , select type Blind and set depth at 71mm ( 65 + 16 -10 ). 

    c. Click on Fillet . Select the 4 edges of the vertical block as shown and set fillet radius at 16mm . Click on    to finish.

    1.4) To create the circular boss:

    a. Click on Back view and select the front surface as sketching plane. Click on Circle and sketch a circle concentric with the round top of the block. Click on Smart Dimension and set the diameter of the circle to 32mm.

    Alternative way

    a. Click on Back view and select the front surface as sketching plane. Click on Perimeter Circle and sketch a circle by clicking on the 3 points as shown

    .

    b. Click on Extruded Boss/Base , select type Blind and set depth at 5mm. Click on     to finish.

    1.5) To create the hole on the circular boss: Select the circular face of the circular boss as the sketching plane.

    a. Click on Circle and sketch a circle concentric with the circular boss. Click on Smart Dimension and set the diameter of the circle to 12mm.

    b. Click on Extruded Cut , select type Through All. Click on    to finish.

       

    1.6) To fillet and create 2 holes on the base :

    a.Click on Fillet.  Select the two edges as shown and set fillet radius at 12mm. Click on    to finish.


    b. Select the top face of the base as sketching plane. Click on Circle    and sketch 2 circles concentric with the curved edges. Click on Smart Dimension and set the diameter of the circles to 12mm.

    c. Click on Extruded Cut , select type Through All. Click on   to finish.






    1.7) To create the rib: First, create the sketching plane. Click on Reference Geometry -> Plane. Select the side wall of the base as the 'First Reference'. Set the Offset Distance to 42mm. Click on   to finish. (Check the Flip box if necessary to ensure that the inserted plane lies in the middle of the solid model).

                   

    a. Click on Right view and sketch a line starting from the edge of the base to somewhere on the vertical block. Click on Smart Dimension and set the vertical distance between the two end points of the line to 40mm.

    b. Click on Rib . Select Both Sides and set Distance to 10mm. (If necessary, select Flip material side to ensure the direction of rib is inwards.) Click on    to finish.

     

    1.8) Refining the bracket :

    a.Click on Fillet . Select the edges as shown and set fillet radius to 5mm. Click on    to finish.


    b.Click on Fillet . Select the edges as shown and set fillet radius to 3mm. Click on    to finish.

           

    1.9) You can now save this model by clicking File -> Save. Name this file as "Bracket".


    Step 2: Creating the Base

    2.1) To create a new document: Click on New  and select   to create a new Part document.

    2.2) To create the base: Select Right plane in the Features Manager. (You need to select a plane before you can sketch.)

    a. Select Line  and roughly sketch the cross-sectional area of the base as shown. 

    b. Click on Smart Dimension  and set the respective dimensions shown below.

    c. Click on Extruded Boss/Base , select type Blind and set depth at 84mm. Click on  to finish.

    d. Click on Fillet . Select the 4 side edges of the base as shown and set fillet radius as 12mm. Click on   to finish.

    e. Click on Fillet . Select the 2 edges at the bottom of the base as shown and set fillet radius as 3mm. Click on    to finish.

     

     

     

    2.3) To create one of the holes in the base: Select the face highlighted as shown as the sketching plane.

    a. Click on Circle   and sketch a circle concentric with the 12mm side fillet. Click on Smart Dimension    and set its diameter to 12mm. Click on   to finish.

       

    b. Click on Extruded Cut , select type Through All. Click on   to finish.

    2.4) To create another 3 holes on the base: Since the 4 holes are arranged in a pattern, making use of Linear Pattern  will be more efficient than to create the additional 3 holes individually. 

    Select the hole created and click on Linear Pattern . Select the edge indicated by the diagram below to set as Direction 1. Click on Reverse Direction  if necessary to ensure the copies are in the correct orientation. Set Number of Instances to 2 and the Spacing as 60mm. Follow the diagram to set the indicated edge as Direction 2. Click on Reverse Direction  if necessary to ensure the copies are in the correct orientation. Set Number of Instances to 2 and the Spacing as 126mm. Click on  to finish.

         

    2.5) You can now save this model by clicking File -> Save. Name this file as "Base".

      

    Step 3: Creating the Roller

    3.1) To create a new document : Click on New and select   to create a new Part document.

    3.2) To create the roller: Select Right plane in the Features Manager. (You have to select a plane before you can sketch).

    a. By using Centerline and Line , sketch the following diagram as shown and use Smart Dimension  to set the appropriate dimensions. (Draw centerline horizontally and vertically across the origin).


    b. Click on Revolved Boss/Base . Select the horizontal center-line as Axis of Revolution, select type Blind and set Angle as 360 degrees. Click on to finish.

       

     

    3.3) Filleting the Roller: Click on Fillet . Select the 4 circumferences on both inner sides of the roller and set the fillet radius to 6mm. Click on to finish.

     

    3.4)  You can now save this model by clicking File -> Save. Name this file as "Roller".

      

     

     

    Step 4: Creating the Spindle

    4.1) To create a new document : Click on New and select   to create a new Part document.

    4.2) To create the body of the spindle: Select Front plane in the Features Manager. (You need to select a plane before you can sketch).

    a. Click on Circle and sketch a circle with the Origin as the center of circle. Click on Smart Dimension and set its diameter to 20mm.

    b. Click on Extruded Boss\Base , select type Blind and set depth at 66mm. Click on   to finish.

     

    4.3) To create the circular boss at both ends of the spindle:

    a. Select one circular end of the spindle body as sketching plane. Click on Circle   and sketch a circle at the center. Click on Smart Dimension and set its diameter to 12mm.

    b. Click on Extruded Boss\Base , select type Blind and set depth at 17mm. Click on   to finish.


    c. Repeat the above procedures for the other circular end of the spindle body.

    4.4) To create the chamfer at the ends: Under Fillet tab, click on Chamfer . Select the two edges highlighted in green as shown. Select Angle distance, and set Distance to 2mm and Angle to 45deg. Click on   to finish.

       

      

     

     

     

    4.5) Creating the drilled hole on the spindle :

    a. Select the front face of the left circular boss created from Step 4.3. Click on Hole Wizard . Under Type tab, for 'Hole Type', select Hole , Standard: Ansi Metric and Type: Drill Sizes. For 'Hole specifications', select Size: Ø6.0. In the End Condition, select Blind and Hole depth 50mm.


    b. Under Positions tab. By moving the cursor, readjust the hole going to be created to concentric with the origin. Click on   to finish.

    4.6) Creating the drilled hole at the top of the spindle:

    a. Click on Reference Geometry -> Plane. Select the Top plane (from the drop down list as shown) as 'First Reference' and set the Distance as 10mm (Ensure the green plane lies above the spindle). Click on   to finish.

       

    b. Click on Hole Wizard . Under Type tab, for 'Hole Type', select Hole , Standard: Ansi Metric and Type: Drill Sizes. For 'Hole specifications', select Size: Ø6.0.  In the End Condition box, select Blind and Hole depth 15mm.


    c. Click on the Positions tab. Place the hole roughly at the center of the spindle. Click on Smart Dimension  and readjust the position of the hole (i.e. 33mm and 10mm from the respective edges of the middle part). Click on  to finish.

     

     

    4.7) You can now save this model by clicking File -> Save. Name this file "Spindle".

     

     

    Step 5: Creating the Bush

    5.1) To create a new document : Click on New and select    to create a new Part document.

    5.2) To create the circular disc : Select Front Plane in the Feature Manager. (You have to select a plane before you can sketch.)

    a. Click on Circle   and sketch a circle of diameter 32mm.

    b. Click on Extruded Boss/Base , select type Blind and set depth at 3mm. Click on    to finish.


    5.3) To create the smaller circular boss :

    a. Select front face of the disc. Click on Circle  and sketch a circle of diameter 25mm concentric with the circular disc.

    b. Click on Extruded Boss/Base , select type Blind and set depth at 22mm. ( 25 - 3 ) Click on    to finish.


    5.4) To create the hole in the bush :

    a. Click on the face of the smaller circular boss to select it as the sketching face. Click on Circle   and sketch a circle of diameter 20mm with the circular face selected.

    b. Click on Extruded Cut , select type Through All. Click on    to finish.

    5.5) You can now save this model by clicking File -> Save. Name this file as "Bush".

    Step 6: Assembly

    6.1) To create a new assembly document: Click on New and select   to create a new Assembly document. Under 'Part/Assembly to Insert', click on Browse and select the file Base.sldprt. This action will set the part Base as the base for the assembly, i.e. it is fixed and cannot be moved. Click on the graphics window to place the base.

    6.2) To import the Bracket into the assembly : Click on , Browse and select the file Bracket .sldprt. Click on the graphics window to place the bracket.

    6.3) To assemble the Brackets with the Base :

    a. Click on Mate . Select the surfaces of the Bracket and Base as highlighted in diagram. Select Coincident . Click on   to finish.


    b. Select the surfaces highlighted in the diagram. Select Coincident . Click on   to finish.


    c. Select the surfaces highlighted in the diagram. Select Coincident . Click on    to finish.


    d. Repeat the procedures in Steps 6.2 -6.3 to assemble the second Bracket on the other side of the Base. (Under 'Move Component' tab, select Rotate Component and rotate the second bracket to the correct direction before mating it with the base.)

    e. Click on   to exit Mate.

    6.4) To import the Spindle into the assembly: Click on Insert Components , Browse and select file Spindle .sldprt. Click on the graphics window to place the Spindle.

    6.5) To assemble the Spindle with the Assembly:

    a. Click on Mate . Select the surfaces of the Bracket and Spindle as highlighted in the diagram. Select Concentric . Click on    to finish.


    b. Select the surfaces highlighted in the diagram. Select Coincident . Click on     to finish.


    c. Click on    to exit Mate.

    6.6) To import the Roller and 2 Bushes into the assembly : Click on Insert Components , Browse and select file Roller .sldprt. Place the Roller away from the assembly. Repeat the procedure to import the 2 Bushes into the assembly file.

    6.7) To assemble the Bushes with the Roller :

    a. Click on Mate . Select the surfaces of the Bush and Roller highlighted in the diagram. Select Concentric . Click on    to finish.


    b. Select the surfaces highlighted in the diagram. Select Coincident . Click on    to finish. Click on    again to exit mate.


    c. Repeat the procedures in Steps 6.7a and 6.7b to assemble the second Bush on the other side of the Roller. (Under 'Move Component' tab, select Rotate Component and rotate the second bush to the correct direction before mating it with the roller.)

    d. Click on    to exit Mate.

      

     

     

     

    6.8) To assemble the Roller-Bush assembly with the Bracket-Spindle-Base assembly :

    a. Click on Mate . Select the surfaces of the Roller-Bush assembly and the Spindle highlighted in the diagram. Select Concentric . Click on    to finish.


    b. Select the surfaces highlighted in the diagram. Select Distance and set the value to 1mm. Click on to finish.


    c. Click on to exit Mate.


     

    6.9) To change color of parts : Right-Click on the respective parts in the Feature Manager. In the drop-down menu, select Edit Color and the Color Properties will appear. Choose your preferred color for the respective part. Click on   to finish.

        

    6.10) You can now save this assembly by clicking File -> Save. Name this file as "Roller Bracket".
