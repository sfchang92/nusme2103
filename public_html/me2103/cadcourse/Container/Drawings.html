Container

Drawings

 Drawings can be produced in SolidWorks for either Parts or Assemblies. 

ADDING PARENT VIEW

4.1) Click on New and select . Click Browse, and open Container_assembly.SLDASM.

4.2) Under Orientation, select Bottom  . Under Scale, set Use Custom scale to 1:5. This is the scale which we are going to use for the whole drawing. Left click on the lower left of the drawing sheet to place the view. Click on to finish.


 

4.3) To set the Tangent Edge style, right-click on the view. From the drop-down menu, go to ‘Tangent Edge’ and select ‘Tangent Edges With Font’.

 

Note:- The first view that has been added into the drawing view shall be the user-defined parent view.
 
 

ADDING PROJECTION VIEWS
4.4) Select the user-defined parent view. The child views shall now be derived from this user-defined parent view using Projected View . Click on Projected View . Move the pointer above and right of the parent view to create the plan view and end elevation view respectively. (Note: You have to click on the drawing sheet to place the views.) Click on to finish.



(Note that this drawing sheet has the properties of 3rd angle projection.)

4.5) To set the Tangent Edge style for the two new views, repeat the procedure in Step 4.3.

ADDING ISOMETRIC VIEWS
4.6) To insert an isometric view, click on Model View . Container-assembly.SLDASM should already be available in the open documents box; double–click to select the file. (If the file is not available, click browse and open the file again.)

4.7) Under the Orientation box, select Isometric  . Place the view as shown. Click on to finish.

4.8) Select the isometric view, right-click, go to 'Tangent Edge' and select 'Tangent Edges With Font'.

ADDING ANNOTATIONS

4.9) To label the views, click on Note on the Annotations toolbar. Move the cursor to the desired location and name the views appropriately.

 

4.10) To add the Angle of Projection, click on Note . Click on Add Symbol . Under Symbol library, select Projection Symbol and choose Third Angle Projection. Click Ok and place it under "Angle of Projection" at the bottom of the drawing sheet.

4.11) Click on Note . A box will appear with the pointer. Click in the 'TUTORIAL NAME' box. Fill in "CT1 CONTAINER" with the appropriate font and size. Click on to finish.

Repeat using Note  to fill in the rest of the boxes with your own name, matriculation no., class and the date.

4.12) The finished drawing sheet will look something like the diagram below. The sample below only acts as a guide. Check with your tutor on what needs to be included in the drawing sheet.


(Ensure that you have typed in your own name and matric no.)


SAVING THE DRAWING
4.11) Click Save  or click File -> Save. Save this file as "Container_drawing".
