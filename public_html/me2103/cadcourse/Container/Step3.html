Step 3: Assembly


3.1) SIMPLE ASSEMBLY MODELING EXERCISE

Based on the basic knowledge on assembly modeling, an assembly could be created with 2 components created in Step 1 and Step 2. Typically, 3 mating relationships are needed to fully constrain the 6 degrees of freedom in an assembly.

3.1.1) Click on New and select .

3.1.2) Click on "Browse" and select the "Cover" solid part saved.

 

3.1.3) Left-click to place the component. The first component added into the assembly would be fixed.

 

INSERTING THE SECOND COMPONENT
3.1.4) Click on Insert Component and "Browse" for the part: Base.SLDPRT. Left click anywhere in the graphics area of the assembly to place the second component. (This second component is not fixed at this moment. It can be moved using Move Component and rotated using Rotate Component .)


　

ADDING THE FIRST MATING RELATION

3.1.5) Click on Mate   and select the pair of faces as shown. Select the Coincident  option. Select to accept the mate relation. (This aligns the two faces together. The second component can be moved freely using the Move Component  but could now rotate about one orientation only using Rotate Component .)


　

ADDING THE SECOND MATING RELATION
3.1.6) Select another pair of faces as shown. Select the Coincident option. Select   to accept the mate relation. (This aligns these two faces together. The second component can be moved in one direction using Move Component  but could not rotate using Rotate Component .)


　

ADDING THE 3RD (AND LAST) MATING RELATION
3.1.7) Next, select the pair of edges as shown. Select the Coincident option. Select   to accept the mate relation. (This aligns these two edges together. The second component cannot be moved using Move Component and rotated using the Rotate Component ). Click on  again. The components are now fully defined in the assembly and all degrees of freedom are constrained.


Note: This is just one suggested method of assembly modeling for this example. There are many other combinations of ways of constraining the parts.
　

SAVING THE ASSEMBLY
3.1.8) Click Save   or click File -> Save. Save this assembly as "Container_assembly".
