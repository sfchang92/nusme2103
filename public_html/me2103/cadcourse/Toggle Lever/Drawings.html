Toggle Lever

Drawings

Required:
a. 3 orthographic views (Hidden details are not required)

    (1) A Front Elevation indicated by the arrow A.

    (2) A Plan View.

    (3) A Side Elevation indicated by the arrow B.

b. Indicate clearly the angle of projection.

CREATING THE FRONT, END & PLAN VIEW
9.1) Click on New and select  to create a new drawing document.

Click on Standard 3 View . Click 'Browse' and go to the location of your "Toggle Lever" part file.

9.2) To set the Tangent Edge style, right-click on the front view, from the drop-down menu, go to ‘Tangent Edge’ and select ‘Tangent Edges Removed’. Do the same for the other 2 views.

9.3) The generated views use the default scale of the sheet i.e. 1:2. To change the scale, so that the drawing views occupy the sheet more fully, click on the Front View. The Drawing View manager appears. In the Scale box, click on 'Use custom scale', and select 1:1.

NB: The other views (End Elevation and Plan View) will automatically scale up to 1:1 because, by default, they follow the scale of the Parent View. In this case, the Front Elevation is the Parent View. 




CREATING THE ISOMETRIC VIEW

9.4) a. Click on Model View . Double-click on the "Toggle Lever" under "Open documents:"

b. Select Isometric  in the Orientation box. In the Scale box, click on Use Custom Scale and select 1:1. Click on the drawing sheet to place the isometric view as shown.


 

CREATING THE CENTER MARKS AND CENTERLINES

9.5) a. Click on Center Mark from the Annotation Toolbar and add the following center marks as shown.

b. Click on Centerline and draw the following centerlines as shown.

CREATING THE ANNOTATIONS AND PROJECTION SYMBOL

9.6) a. Click on Note from the Annotation Toolbar. Label the views.


b. To add the projection symbol, click on Note , select Add Symbol    and under Symbol library, select Projection Symbols. Then, select Third Angle Projection and place it under "Angle of Projection".



9.7) You can now save this file by clicking File -> Save.
