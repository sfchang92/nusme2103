Step 2: Tangent Line

2.1) Sketch a Line on the left of the circles as shown. Press Esc to deactivate the Line tool.

2.2) Hold Ctrl and select the line and the top outer circle with which the line is to be tangent to. Ensure that only the line and the circle have been selected. Under "Add Relations", select Tangent  and click on to finish. (For every successful application of the tangent relation, the symbol  appears.)

NB: 'Ctrl' allows the simultaneous selection of multiple objects.

               

2.3) Hold Ctrl and select the line and the bottom outer circle. Select Tangent again. Click on to finish.

2.4) Trim the line by using Trim to obtain the following as shown. (Note:- Under Options in the Trim Properties Manager, select ‘Trim to closest’    ).
