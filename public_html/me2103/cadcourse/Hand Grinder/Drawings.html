Hand Grinder

Drawings

Required:

a) a fully assembled sectional front elevation on A-A corresponding to the front elevation of the base.

b) a plan view of the assembly

c) an end elevation looking in the direction of arrow B

d) an isometric view

CREATING THE PLAN VIEW

Before you begin, right click on the top of the window and check the Sketch toolbar and the Annotations toolbar. Click and Drag them to the side of the window. This allows us to access the tools easily.

7.1) Click on New and select  to open a new drawing file from the ME2103 template. Click Browse, and open Hand Grinder.SLDASM.

7.2) Under "Orientation", select Front . Set the "Custom scale" to 1:4. This is the scale which we are going to use for the whole drawing. Left click on the desired location of the drawing sheet to place the view. Click  to finish.

   

To set the Tangent Edge style, right-click on the view. From the drop-down menu, go to ‘Tangent Edge’ and select ‘Tangent Edges Removed’.

 

ADDING THE SECTION VIEW A-A

7.3) Click on Section View . Select Vertical cutting line  and left click to place the cutting plane such that it passes through the center.

Click on  and the section view box will appear. Make sure you check the auto hatching box. In the Excluded Components box, select the Rid of Housing, Keys, Nuts and Washers. Click ok.


 

 

**Changing the line style: 

Select Options:

Click on Document Properties -> View Labels -> Section.
Under 'Line Style', select the last line style from the drop down list. Click Ok.

ADDING THE END ELEVATION VIEW

7.4) Click on Projected View . When prompt to select, click on the drawing of Section View A-A. Move the pointer to the left side (to get an end elevation view from direction B and left-click to place the view. Click to finish.). To set the Tangent Edge style, right-click on the section view and end elevation view to set ‘Tangent Edges Removed’.

 
ADDING THE ISOMETRIC VIEW

7.5) Click on Model View . Double-click on the Hand Grinder document. Under "Orientation", select Isometric and set "Custom Scale" to 1:3. Place the isometric view as shown. To set the Tangent Edge style, right-click on the isometric view. From the drop-down menu, go to ‘Tangent Edge’ and select ‘Tangent Edges Removed’.

     

ADDING ANNOTATIONS

7.6) To create the centerlines using Centerline   from the Annotations Toolbar (not this ):

Before creating the centerlines, click on the plan view and under Display Style, select Hidden Lines Visible. Do the same for the end elevation view.

Click on Centerline and under Auto Insert, check Select View.

Select the plan view, end elevation view and sectional view A-A. Click on  to finish.

After creating the centerlines, click on the plan view and under Display Style, select back Hidden Lines Removed. Do the same for the end elevation view.

Centerlines created on the three views:
*Right click and hide unnecessary hidden lines created.



7.7) Select Center Mark  from the Annotations Toolbar. Select the respective features on the drawings to add in the center marks as shown.

ADDING DIMENSIONS

7.8) Use Smart Dimension  to add the 5 dimensions indicated with an *. (Deselect Add Parenthesis  to remove the parenthesis.)

LABEL THE VIEWS AND FILL IN TABLE

7.9) Click on Note   to add/edit the labels of the respective views. Fill in your Name, Matric number, Class, Scale, Date, Tutorial Name and Comments. 

 


LABEL SYMBOL OF PROJECTION ANGLE

7.10) Click on Note   and Add Symbol. Selection the correct projection symbol from symbol library.

 

7.11) You can now save this file by clicking File -> Save.
