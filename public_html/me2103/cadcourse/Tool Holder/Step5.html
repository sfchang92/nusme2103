STEP 5: Creating the Chamfer

 

5.1) To create the chamfer: Select the edge as shown. Click on Chamfer . Select Flip direction, and define Distance as 14mm and Angle as 30degrees.


5.2) You can now save this file by clicking File -> Save. Save this model as "Tool Holder".
