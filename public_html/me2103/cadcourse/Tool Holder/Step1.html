STEP 1: Creating the Body

1.1) Click on New and select  to create a new Part document.

1.2) Select Front plane. (You have to select a plane before you can sketch.)

1.3) Select Rectangle and draw a rectangle.

1.4) Click on Smart Dimension and set the breadth to 40mm and length to 65mm.


1.5) Click on Extruded Boss/Base and set the depth to 100mm. Click on to finish.
