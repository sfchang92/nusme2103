Tool Holder

Drawings

 

Required:

a. 3 orthographic views (Hidden details are not required)

(1) A Front Elevation indicated by the arrow A.

(2) A Plan View.

(3) A Side Elevation indicated by the arrow B.

b. Indicate clearly the angle of projection.

6.1) To create a new drawing document : Click on New and select  .

6.2) To import the tool holder file : Click on Browse and select the Tool Holder part saved in Step 5.2

6.3) To add a front view : Under Orientation, select Right . Under Scale, select Use Custom Scale, set scale as 1 : 1. Position the view as shown. Click on or Press Escape on keyboard to finish.

 

6.4) To add a side elevation and plan view : Click on Projected View . Move the cursor and click on the drawing sheet to add the side elevation and plan view as shown. Click on .


 

6.5) To add the isometric view :

a. Click on Model View or click on Insert -> Drawing View -> Model. Double-click on the "Tool Holder" under "Open documents:".
 

b. Under Orientation, select Isometric , under Display Style, select Hidden lines removed, under Scale, select Use Custom Scale, set scale as 1 : 1. Position the view as shown. Click on or Press Escape on keyboard to finish.

 

 

 

 

6.6) To change Tangent Edge style : Right-click on the Side Elevation View. From the drop-down window, go to Tangent Edge and select Tangent Edges With Font.

 

6.7) To add the Annotations :

a. Click on Center Mark  from the Annotation toolbar. Select the holes and circular extrusions from the Plan and Front view to add the required Center Marks.

 

 

Click on Centerline and draw the following center lines as shown.






b. Click on Note  from the Annotation toolbar. Label the views.

 

c. To add the 3rd Angle Projection Symbol, click on Note  and select Add Symbol . In the Symbol dialog box, under symbol library, select Projection Symbols. Then, select Third Angle Projection and place it under "Angle of Projection".

 

 

6.8) You can now save this file by clicking File -> Save.
