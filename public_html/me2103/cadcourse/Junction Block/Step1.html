Junction Block

Step 1: Creating the Body

1.1) To create a new document: Click on New and select   to create a new Part document.

1.2) To create a rectangular solid: Select Right plane in the Features Manager and click on Sketch  .  

a. Select Rectangle and sketch a rectangle of any size. Click on Smart Dimension , change the length of the rectangle to 74mm ( 12 + 25 + 25 +12 ) and the breadth to 109mm ( 76 + 48 + 23 - 38 ).

b. Click on Extruded Boss/Base , select type Blind and set depth at 64 mm. Click on   to finish.

   

 

   

 

 

 

 

 

 

 

1.3) To create the curved top : Click on Fillet . Set the fillet radius to 37 mm. Select the two edges as shown. Click on   to finish.

 

         

 

 

 

 

1.4) To create the shell : Click on Shell and set the Thickness to 12mm. Select the front face of the model. Click on to finish.
