Junction Block

Drawings

Required:

1. Add 2 sectional views as indicated by the cutting planes X-X and Y-Y.

2. Add a plan view as indicated by the arrow E.

3. Add 5 dimensions indicated with *.

4. Indicate clearly the angle of projection that you are using.

8.1) To create a new drawing document : Click on New    and select     to create a new drawing.

8.2) To import the Junction Block file : Under 'Part/Assembly to Insert', click on Browse and select the Junction Block part saved in Step 7.3.

8.3) To add a Model view : Under 'Orientation', select Right   . Under 'Display Style', select Hidden lines removed. Under  'Scale', select Use Custom Scale and set scale as 1 : 2. Position the first view as shown. Click on or Press Escape on keyboard to finish.


8.4) To add a Projected view : Click on Projected view , position the second view as shown and click on the drawing sheet to place the view. Click on or press Esc on keyboard to finish.

 

 

8.5) To create section view X-X: Click on the second view to set it as the active view. Click on Section View . Ensure that the cutting line is vertical. Position the cutting plane such that it passes through the center as shown and click to place it on the view. 

 Click on  and a Section Scope window will appear. Click on the  beside Drawing View 2. Select all the Ribs from the drop down list to exclude hatching. Ensure that the cutting is in the correct direction. 

Click Ok. Place the sectional view as shown.

Change the alphabet to X and click on.

8.6) To create section view Y-Y : Click on the first view to set it as the active view. Click on Section View . Ensure that the cutting line is vertical. Place the cutting plane such that it cuts through the center as shown.

Click on and a Section Scope window will appear. Select all the Ribs to exclude hatching. Check Flip direction and ensure that the cutting is in the correct direction. Click Ok and place the sectional view as shown.

Change the alphabet to Y and click on.

8.7)  Changing the line style : 

Select Options:

Click on Document Properties -> View Labels -> Section.
Under 'Line Style', select the last line style from the drop down list. Click Ok.



8.8) To create the Plan View : 

Select the first view and click on Projected View . Place the plan view as shown and click ,

8.9) Rearranging the views : Right click on the first view and select Hide. When prompted whether to hide dependent views, click No. Repeat the same steps to hide the second view. Right click the sectional view X-X, go to Alignment and click on Break Alignment. Do the same for sectional view Y-Y and the plan view. Reaarange the views as shown.

Next, add an Isometric View. Click on Model View and repeat the procedures in step 8.2 and 8.3 except under 'Orientation', select Isometric. Place the Isometric View as shown. Click .

8.10) Editing tangent edges : Right click on the Plan view, go to Tangent Edge and select Tangent Edges with Font. Do the same for the Isometric view. As for the remaining sectional views, select Tangent Edges Removed.

 

8.11) Adding center marks and centerlines : Click on Center Mark. Select the 4 holes on the plan view and the holes on the sectional views respectively. Click .

Select sectional view Y-Y and sketch the centerlines accordingly as shown.

Sketch the centerlines for the remaining 2 views.

8.12) Adding cutting planes on the sectional views : Repeat steps 8.5 and 8.6 to create the respective cutting planes. Place the created sectional views outside of the drawing sheet and hide them.

8.13) Add the 5 dimensions indicated with * : Use Smart Dimension  to create the following dimensions. (Note: Under 'Dimension Text', uncheck Add parenthesis  for each dimension.)

For the dimension of the holes on the plan view: Click Note  , under 'Leader', select Bent Leader and type the following text. Click .

(To insert the diamater symbol, under 'Text Format', Click on Add Symbol and select Diameter.)

For the Square Base dimension: Click Note and select the bottom left corner of the sectional view Y-Y. Type "SQ.BASE" and select Leader Right and Underlined Leader. Click .


 8.14) Adding annotations : Click Note and insert/edit the respective notes.

Under 'Comments' and 'Scale' at the bottom of the drawing sheet, add the following note as shown.

8.15) Indicating the angle of projection : Click Note , under 'Text Format', click on Add Symbol.

Under 'Symbol library', select Projection Symbol from the drop down list. Select Third Angle Projection and click Ok.

Place under 'Angle of Projection'. Click .

8.16) Ensure that you have typed in your name and matric no. You can now save this Drawing by clicking File -> Save.
