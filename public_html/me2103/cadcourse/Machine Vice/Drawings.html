Machine Vice

Drawings

Required:

a) a fully assembled sectional front elevation on A-A corresponding to the front elevation of the base.

b) a plan view of the assembly

c) an end elevation looking in the direction of arrow B

d) an isometric view

CREATING THE PLAN VIEW

Before you begin, right click on the top of the window and check the Sketch toolbar and the Annotations toolbar. Click and Drag them to the side of the window. This allows us to access the tools easily.

6.1) Click on New and select  to open a new drawing file from the ME2103 template. Click Browse, and open Machine Vice.SLDASM.

6.2) Under "Orientation", select Top . Set the "Custom scale" to 1:2. This is the scale which we are going to use for the whole drawing. Left click on the desired location of the drawing sheet to place the view. Click  to finish.

        

To set the Tangent Edge style, right-click on the view. From the drop-down menu, go to ‘Tangent Edge’ and select ‘Tangent Edges Removed’.

    

6.3) However, a rotated view of the plan view is preferred (so as to create section A-A). Right click on the drawing view --> Select Zoom/Pan/Rotate ---> Select Rotate View. In the Drawing View Angle, type -90deg to rotate clockwise. (Typing 90deg will rotate the view anti-clockwise.) Click on Apply.


ADDING THE SECTION VIEW A-A

6.4) Click on Section View . Select Horizontal cutting line   and left click to place the cutting plane such that it passes through the center as shown.

Click on  and the section view box will appear. Make sure you check the auto hatching box. In the Excluded Components box, select the Screw, Washer and 2 Hexagonal nuts. Click ok.

 

6.5) In the Section View manager, tick the Flip direction box if necessary to correct the direction of the sectioning view. Make sure that auto hatching is ticked so that hatch lines of different orientation are applied to the different components. Change the section label to A and left-click on the drawing sheet to place the section view below the first view as shown. Click on  to finish. (Note: You can move the views by left-clicking on them and dragging them.)

       

**Changing the line style: 

Select Options:

Click on Document Properties -> View Labels -> Section.
Under 'Line Style', select the last line style from the drop down list. Click Ok.

ADDING THE END ELEVATION VIEW

6.6) Click on Projected View . When prompt to select, click on the drawing of Section View A-A. Move the pointer to the left side (to get an end elevation view from direction B and left-click to place the view. Click to finish.). To set the Tangent Edge style, right-click on the section view and end elevation view to set ‘Tangent Edges Removed’.


 

ADDING THE ISOMETRIC VIEW

6.7) Click on Model View . Double-click on the Machine Vice document. Under "Orientation", select Isometric and set "Custom Scale" to 1:2. Place the isometric view as shown. To set the Tangent Edge style, right-click on the isometric view. From the drop-down menu, go to ‘Tangent Edge’ and select ‘Tangent Edges Removed’.

     

ADDING ANNOTATIONS

6.8) To create the centerlines using Centerline   from the Annotations Toolbar (not this ):

Before creating the centerlines, click on the plan view and under Display Style, select Hidden Lines Visible. Do the same for the end elevation view.

Click on Centerline and under Auto Insert, check Select View.

Select the plan view, end elevation view and sectional view A-A. Click on  to finish.

After creating the centerlines, click on the plan view and under Display Style, select back Hidden Lines Removed. Do the same for the end elevation view.

Centerlines created on the three views:
*Right click and hide unnecessary hidden lines created.

6.9) To display the hidden details. Right click on the Plan view and go to Properties --> select Show Hidden Edges. Click on the base, moving jaw and screw in the plan view drawing and click OK.


6.10) Similarly, right click on the End elevation drawing and go to Properties --> select Show Hidden Edges. Click on the base in the End elevation view drawing.

6.11) Select Center Mark  from the Annotations Toolbar. Select the respective features on the drawings to add in the center marks as shown.

ADDING DIMENSIONS

6.12) Use Smart Dimension  to add the 5 dimensions indicated with an *. (Deselect Add Parenthesis  to remove the parenthesis.)

LABEL THE VIEWS AND FILL IN TABLE

6.13) Click on Note   to add/edit the labels of the respective views. Fill in your Name, Matric number, Class, Scale, Date, Tutorial Name and Comments. Remember to add the 3rd Angle Projection Symbol too.

6.14) You can now save this file by clicking File -> Save.
