    • Step 1: Creating the Base

    1.1) Click on New and select   to create a new part document

    1.2) Select the Right plane (you have to select a plane before you can sketch). Click on Sketch . 

     

    CREATING THE MAIN BODY

    1.3) Using the Line and Smart Dimemsion tool, sketch the profile as shown with its dimensions.

    1.4) Click on Extruded Boss/Base and type Depth 100 mm. Click on  .

    1.5) Select the the top surface as shown

     

    1.6) Click on Normal To . Sketch a Rectangle of length 149 mm (200-35-3-13=149) and breadth 38 mm as shown.

                

    1.7) Click on Extruded Cut . Select type Through all. 

    CREATING THE HANDLE AT THE SIDE

    1.8) Click on Right view. Select the face and sketch a Rectangle as shown.

    1.9) Click on Extruded Boss/Base and type Depth 38 mm. (20+18=38)

    1.10) Click on Fillet . Select the edges of the handle as shown and set fillet radius to 18 mm. Click on  to finish.

       

     1.11) Select the top face of the handle. Click on Normal To .

        

    1.12) Select Straight Slot . Click on the first and second points as shown to create the slot. Select any point as the third point and use Smart Dimension  to set the width to 28 mm.

     

    1.13) Click on Extruded Boss/Base and type Depth 3mm. Click on . Select the top surface of the boss. Click on Sketch . Select Offset Entities. Set offset distance to 7 mm. Check the 'Reverse' direction box to ensure that the offset is inwards. Click on .

    (Note: Ensure that you select Sketch  first before selecting Offset Entities .)
     

                

    1.14) Click on Extruded Cut . Select Through All and click on to finish.

     

    MIRRORING THE HANDLE

    1.15) Select the face as shown. Click on Reference Geometry and select Plane. Set offset distance 50mm and select the 'Flip' box if necessary to ensure the plane lies in the middle of the model. Click on .

           

    1.16) Click on Mirror  with the plane as the Mirror face/plane. Click Features to Mirror box and select the 4 main features of the handle to mirror: the extruded block, the extruded boss on top of the block, the cut-extrude and the fillet.

             

      

    CREATING THE HOLES

    1.17) Click on the Front  view and select the face. Sketch a Circle  with the dimensions shown. Click Extruded Boss/Base and type Depth 3mm.

           

    1.18) Click on the face of the extruded boss. Sketch a concentric Circle   of diameter 20 mm. Click Extruded Cut . Select type Up to Next. Click on .

           

    1.19) Click on the Back view and select the face. Click on Front view and sketch a circle of diameter 26 mm with its center coincident with the center of the hole created just now. Click on Extruded Boss/Base and type Depth 3mm. Click on .

    1.20) Click on Back view, select the face of the extruded boss and sketch a concentric circle of diameter 13 mm. Click Extruded Cut . Select type Up to Next. Click on to finish.

     

    1.21) We now need to create the tap holes (M5 x 0.8) using the Hole Wizard tool. 

    Click on Hole Wizard . Under Type tab, select Straight Tap , Standard Ansi Metric, Type Bottoming Tapped Hole and Size M5x0.8. Under Options, select Cosmetic thread .Under End Condition and under "Thread", set Tap Thread Depth 13 mm. The Blind Hole Depth will automatically change to 15.40 mm.

    Note: Select Cosmetic thread first before setting the "End Condition" to avoid the problem of "End Condition" reverting back to default value.

      

    1.22) Select the Positions tab. Click on Back view. Select the face where the two holes are to be created. Left click to create the holes and use the Smart Dimension tool to position the holes as shown. Click on to finish.

       

     

    FILLET THE MODEL

    1.23) Click on Fillet and select the edge as shown. Set fillet radius to 13 mm. Click on to finish.

    1.24) Click on Fillet and select the edges of the front and back bosses. Set fillet radii to 3 mm. Click on to finish.

    1.25) Click on Fillet and select the 4 inner edges of the handles. Set fillet radii to 20 mm. Click on to finish.

    1.26) Click on Fillet and select the 2 extruded boss edges as shown. Set fillet radii to 3 mm. Click on to finish.

    1.27) You can now save this file by clicking File -> Save. Name this solid model as "Base".

    Step 2: Creating the Moving Jaw

    2.1) Click on New and select  to create a new part document

    2.2) Select the Right plane (you have to select a plane before you can sketch). Click on Sketch . 

    2.3) Select Line and sketch the profile with the dimensions as shown

    2.4) Click on Extruded Boss/Base . Set Depth to 52 mm.

    2.5) Click on Right view and select the face. Sketch a Rectangle as shown.

    2.6) Click on Extruded Cut . Set Depth to 3 mm.

    CREATING THE HOLES

    2.7) Click on Hole Wizard . Select Straight Tap , Standard Ansi Metric, Type Bottoming Tapped Hole and Size M5 x 0.8. Under Options, select Cosmetic thread . Under End Condition, under "Thread", set Tap Thread Depth to 13 mm (The Hole Depth will be automatically changed to 15.40mm).

        

    2.8) Select the Positions tab. Click on Right view. Select the face where the two holes are to be created. Left click to create the holes and use the Smart Dimension tool to position the holes as shown. Click on to finish.

    2.9) For the hole M20 SQ. TH., because SolidWorks is unable to generate screw holes with square threads, we model the hole using an extruded cut. Click on the desired surface and sketch a Circle  of diameter 20mm as shown.

    2.10) Click Extruded Cut  . Select Type Through all. Click on to finish.

     

    FILLET THE JAW

    2.11) Click on Fillet . Select the edge of the jaw as shown and set fillet radius to 13 mm.

     

    2.12) You can now save this file by clicking File -> Save. Save this solid part as "Moving Jaw".

    Step 3: Creating the Screw

    3.1) Click on New  and select    to create a new part document.

    3.2) Select the Right plane (you have to select a plane before you can sketch). Click on Sketch . 

    3.3) Sketch 2 Rectangles  with the dimensions as shown. (Ensure that both rectangles lie on the same line. If necessary, add "Collinear" relation between the bottom two lines of both rectangles using Add Relation . )

           

    3.4) Sketch a Circle of diameter 20 mm below the two rectangles. Click on Add Relation . Select the bottom right point of the left rectangle and the circle. Click on Coincident . Click on to finish.

         

    3.5) Repeat Add Relation  with the same circle but with the bottom left point of the other rectangle. Click on Coincident . Click on to finish.

       

    3.6) Click on Trim Entities . Select Trim to closest  . Trim away the unwanted arc on the circle to obtain the desired profile.

                        

    3.7) Click on Centerline . Create a Centerline as shown. Exit the sketch tool by clicking on .

    3.8) Click on Revolved Boss/Base .  Select the centerline as the Axis of Revolution. In the Selected Contours, click on the rectangles and the area in between, one at a time. You should have three selected regions as shown below. Click on   to generate the screw head.

         

     

    3.9) Click on Back view. Select the face and sketch a concentric Circle  of diameter 20 mm. Click on Extruded Boss/Base and type Depth 190 mm (38+152). Click on to finish.

    (Note: In SolidWorks, we cannot create external square threads on the screw. Hence that part is modeled as an ordinary extrude. We can still add annotations later to identify the part with square threads.)

       

    3.10) Click on Back view. Select the face and sketch a concentric Circle of diameter 13 mm. Click on Extruded Boss/Base and type Depth 16 mm.

           

    3.11) Click on Back view. Select the face and sketch a concentric Circle of diameter 10 mm. Click on Extruded Boss/Base   and type Depth 20 mm.

            

     

    CREATING THE HOLE ON THE SCREW HEAD

    3.12) We can use the Hole Wizard to create the hole easily. But before that, we need 2 reference planes for dimensioning of the hole later.

    a. Click on Reference Geometry --> Plane. Click on  in the model window. Select Right as shown below. Set Offset Distance 18 mm. Click on  to finish. 

       

    b. Click on Reference Geometry --> Plane. This time, select Top and set Offset Distance 36 mm. Click on to finish. 

          

     

    3.13) Select Plane 2 (the horizontal one). Click on Hole Wizard . Select Hole  , Standard Ansi Metric, Type Drill Sizes and Size 10.0. Under End Condition, select Blind and Hole depth 36 mm.

      

    3.14) Select the Positions tab. Click on Top  view. Left click on the middle of the screw head as shown. Click on Smart Dimension .

    The vertical distance can be easily positioned by clicking on the bottom edge of the head and the hole center.

    For the horizontal distance, use Plane 1 for help in dimensioning. Click  to close the wizard.

                 

     

    FILLET THE HEAD

    3.15) Click on Fillet and select the edge of the screw head as shown. Set fillet radius to 3 mm. Click on .

        

     

    3.16) You can now save this file by clicking File -> Save. Save this solid part as "Screw".

    Step 4: Creating the Jaw Grip

    4.1) Click on New and select  to create a new part document

    4.2) Select the Right plane (you have to select a plane before you can sketch). Click on Sketch . 

    4.3) Sketch a Rectangle  with the dimensions as shown.

    4.4) Click on Extruded Boss/Base and type Depth 6 mm.


     

    CREATING THE COUNTERSUNK HOLES

    4.5) Click on Hole Wizard . Select Countersink , Standard Ansi Metric, Type Flat Head Screw, Size M5 and Fit Normal. Under End Condition, select Through All.

     

    4.6) Select the Positions tab. Click on Right view and select the face. Left click to create two holes as shown. Use Smart Dimension  to position the holes properly. Click on   to close the wizard.

    4.7) You can now save this file by clicking File -> Save. Save this solid part as "Jaw Grip".

    Step 5: Assembly

    5.1) Click on New and select  to create a new assembly.

    5.2) Under "Part/Assembly to Insert", click on Browse and select base.SLDPRT. Click anywhere on the right window to place the part. This action will set the part "Base" as the base for the assembly, i.e. it is fixed and cannot be moved.
    5.3) To impart the Jaw Grip into the assembly: Click on Insert Component , Browse and locate jaw_grip.SLDPRT. Click in the right window to place the jaw grip.


    ASSEMBLY OF THE JAW GRIP ONTO THE BASE

    5.4) a. Click on Mate . Select the surface on the jaw grip as shown and the top surface of the kerb on the base. Select Coincident . Click on    when done.
    (Note: Click on Rotate View , Zoom In/Out , Zoom to Area and/or Pan to rotate, zoom in/out and/or move the model to aid in the selection of the faces.)

             

    b. Select the main face of the jaw grip and the corresponding face on the base as shown. Ensure that Coincident  has been selected. Click on when done. (Make sure the countersunk holes are facing outside; we need to add screws later.)
             

    c. Select the edge on the jaw grip and the edge on the base as shown. Ensure that Coincident  has been selected. Click on when done. Click on  again to exit Mate.


           

     

     

    ASSEMBLY OF M5 SCK SCREW ONTO THE GRIP (WITH BASE)

    5.5) a. Click on Insert Component , Browse and locate M5 CSK.SLDPRT.  Click in the right window to place the M5 CSK.

    (The M5 CSK.SLDPRT file should have been given to you via email, IVLE, etc. If not, click on M5 CSK.SLDPRT to get a copy and save it in the same folder as your assembly)

    b. Click on Mate . Select the cylindrical face on the screw and the face of the cut-extrude (hole) on the grip. Select Concentric . (Not Coincident!) Click on . (Select Aligned   if necessary to align the screw in the correct direction.)

           

    c. Click on the countersunk surface of the grip and the revolved base of the screw. Select Coincident . Click on . Click on again to exit Mate.

            

    d. Repeat Steps 5.5a, 5.5b and 5.5c for the other hole. (The Insert Component  allows multiple instances of the same part to be added into the assembly.)

     

     

    ASSEMBLY OF MOVING JAW AND JAW GRIP

    5.6) a. Click on Insert Component , Browse and locate moving_jaw.SLDPRT. Click in the right window to place the moving jaw.

    b. Click on Insert Component , Browse and locate jaw_grip.SLDPRT. Click in the right window to place the jaw grip.

    c. The following steps are similar to those outline in Step 5.4. 

    Click on Mate . Select the surface on the jaw grip which will be in contact with the kerb of the moving jaw. (Make sure the countersunk holes do not face the moving jaw; we need to add screws later). Select the top surface on the kerb at the base. Select Coincident . Click on  when done.
                 

    d. Select the main face of the jaw grip and the corresponding face on the moving jaw as shown. Ensure that Coincident  has been selected. Click on  when done.

              

    e. Select the edge on the jaw grip and the edge on the moving jaw as highlighted below. Ensure that Coincident  has been selected. Click on  when done. Click on again to exit Mate.

     

     

    ASSEMBLY OF M5 CSK SCREW ONTO THE GRIP (WITH MOVING JAW)

    5.7) We need to attach two M5 CSK screws to the jaw grip. This step is exactly the same as Step 5.5. Refer to Step 5.5 on how to attach the screws.

     

    ASSEMBLY OF MOVING JAW SUB-ASSEMBLY AND BASE SUB-ASSEMBLY.

     5.8) a. Click on Mate .  Select the surface along the length of the groove of the base. Select the surface on the side of the jaw. Select Coincident . Click on when done.
               

    b. Select the top surface of the base and the bottom surface of the head of the jaw. Ensure that Coincident  has been selected. Click on  when done. (With 2 degrees of freedom fixed, the moving jaw is free to slide in the groove.) 

      

    c. As stated, we need to fix the distance between the moving jaw and the base. Select one edge from the grip attached to the base, and another edge from the grip attached to the jaw. Do not select Coincident or any relation. Instead, click on the Distance  and type Distance 50 mm. Click on when done. Click on  again to exit Mate.

           

     

     

    ASSEMBLY OF SCREW

    5.9) a. Click on Insert Component , Browse and locate screw.SLDPRT. Click in the right window to place the screw.

    b. Click on Mate . Select the long cylindrical surface of the screw. Select the surface of the hole on the base. Ensure that Concentric is selected. Click on when done.

        

    c) Select the face of the screw as shown. Select the face of the boss on the base. Ensure that Coincident  is selected. (With these 2 conditions, the screw stays in the assembly but is able to rotate.) Click on   when done. Click on  again to exit Mate.

              

    Remarks: Ensure that the hole on the "screw" is facing upward vertically so that the drawing will appear correctly. (Use "Rotate Component"  if necessary.)

     

     

    ASSEMBLY OF WASHER AND NUTS

    5.10) a. Click on Insert Component , Browse and locate Washer.SLDPRT. This should have been given to you. Click in the right window to place the washer.

    (The Washer.SLDPRT file should have been given to you via email, IVLE, etc. If not, click on Washer.SLDPRT to get a copy and save it in the same folder as your assembly)

    b. Click on Mate . Select the cylindrical surface of the screw at the end of the base. Select the cylindrical surface of the hole in the washer. Ensure that Concentric  is selected. Click on when done.

        

    c. Select the face of the washer as shown. Select the face of the boss on the base. Ensure that Coincident  is selected. (With these 2 conditions, the washer stays in the assembly but is able to rotate.)  Click on when done. Click on  again to exit Mate.

     

     

    d. Click on Insert Component , Browse and locate M10 HEXnut.SLDPRT. This should have been given to you too. Click in the right window to place the nut.

    (The M10 HEXnut.SLDPRT file should have been given to you via email, IVLE, etc. If not, click on M10 HEXnut.SLDPRT to get a copy and save it in the same folder as your assembly.)

    e. Click on Mate . Select the cylindrical surface of the screw at the end of the base. Select the cylindrical surface of the hole in the nut. Ensure that Concentric  is selected. Click on when done.

            

    f. Select the face of the nut as shown. Select the face of the washer. Ensure that Coincident is selected. (With these 2 conditions, the nut stays in the assembly but is able to rotate.) Click on when done. Click on  again to exit Mate.

       

     

    g. Repeat Steps 5.10d, 5.10e and 5.10f for the second nut.

    h. To align the nuts. Click on Mate . Select the faces of the nuts as shown. Select Coincident . Click on   to finish. Click on  again to exit Mate.

           

     

     

    5.11) You can now save this file by clicking File -> Save. Save this assembly as "Machine Vice".

    Drawings

    Required:

    a) a fully assembled sectional front elevation on A-A corresponding to the front elevation of the base.

    b) a plan view of the assembly

    c) an end elevation looking in the direction of arrow B

    d) an isometric view

    CREATING THE PLAN VIEW

    Before you begin, right click on the top of the window and check the Sketch toolbar and the Annotations toolbar. Click and Drag them to the side of the window. This allows us to access the tools easily.

    6.1) Click on New and select  to open a new drawing file from the ME2103 template. Click Browse, and open Machine Vice.SLDASM.

    6.2) Under "Orientation", select Top . Set the "Custom scale" to 1:2. This is the scale which we are going to use for the whole drawing. Left click on the desired location of the drawing sheet to place the view. Click  to finish.

            

    To set the Tangent Edge style, right-click on the view. From the drop-down menu, go to Tangent Edge and select Tangent Edges Removed.

        

    6.3) However, a rotated view of the plan view is preferred (so as to create section A-A). Right click on the drawing view --> Select Zoom/Pan/Rotate ---> Select Rotate View. In the Drawing View Angle, type -90deg to rotate clockwise. (Typing 90deg will rotate the view anti-clockwise.) Click on Apply.


    ADDING THE SECTION VIEW A-A

    6.4) Click on Section View . Select Horizontal cutting line   and left click to place the cutting plane such that it passes through the center as shown.

    Click on  and the section view box will appear. Make sure you check the auto hatching box. In the Excluded Components box, select the Screw, Washer and 2 Hexagonal nuts. Click ok.

     

    6.5) In the Section View manager, tick the Flip direction box if necessary to correct the direction of the sectioning view. Make sure that auto hatching is ticked so that hatch lines of different orientation are applied to the different components. Change the section label to A and left-click on the drawing sheet to place the section view below the first view as shown. Click on  to finish. (Note: You can move the views by left-clicking on them and dragging them.)

           

    **Changing the line style: 

    Select Options:

    Click on Document Properties -> View Labels -> Section.
    Under 'Line Style', select the last line style from the drop down list. Click Ok.

    ADDING THE END ELEVATION VIEW

    6.6) Click on Projected View . When prompt to select, click on the drawing of Section View A-A. Move the pointer to the left side (to get an end elevation view from direction B and left-click to place the view. Click to finish.). To set the Tangent Edge style, right-click on the section view and end elevation view to set Tangent Edges Removed.


     

    ADDING THE ISOMETRIC VIEW

    6.7) Click on Model View . Double-click on the Machine Vice document. Under "Orientation", select Isometric and set "Custom Scale" to 1:2. Place the isometric view as shown. To set the Tangent Edge style, right-click on the isometric view. From the drop-down menu, go to Tangent Edge and select Tangent Edges Removed.

         

    ADDING ANNOTATIONS

    6.8) To create the centerlines using Centerline   from the Annotations Toolbar (not this ):

    Before creating the centerlines, click on the plan view and under Display Style, select Hidden Lines Visible. Do the same for the end elevation view.

    Click on Centerline and under Auto Insert, check Select View.

    Select the plan view, end elevation view and sectional view A-A. Click on  to finish.

    After creating the centerlines, click on the plan view and under Display Style, select back Hidden Lines Removed. Do the same for the end elevation view.

    Centerlines created on the three views:
    *Right click and hide unnecessary hidden lines created.

    6.9) To display the hidden details. Right click on the Plan view and go to Properties --> select Show Hidden Edges. Click on the base, moving jaw and screw in the plan view drawing and click OK.


    6.10) Similarly, right click on the End elevation drawing and go to Properties --> select Show Hidden Edges. Click on the base in the End elevation view drawing.

    6.11) Select Center Mark  from the Annotations Toolbar. Select the respective features on the drawings to add in the center marks as shown.

    ADDING DIMENSIONS

    6.12) Use Smart Dimension  to add the 5 dimensions indicated with an *. (Deselect Add Parenthesis  to remove the parenthesis.)

    LABEL THE VIEWS AND FILL IN TABLE

    6.13) Click on Note   to add/edit the labels of the respective views. Fill in your Name, Matric number, Class, Scale, Date, Tutorial Name and Comments. Remember to add the 3rd Angle Projection Symbol too.

    6.14) You can now save this file by clicking File -> Save.
