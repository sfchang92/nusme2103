Step 2: Creating the Moving Jaw

2.1) Click on New and select  to create a new part document

2.2) Select the Right plane (you have to select a plane before you can sketch). Click on Sketch . 

2.3) Select Line and sketch the profile with the dimensions as shown

2.4) Click on Extruded Boss/Base . Set Depth to 52 mm.

2.5) Click on Right view and select the face. Sketch a Rectangle as shown.

2.6) Click on Extruded Cut . Set Depth to 3 mm.

CREATING THE HOLES

2.7) Click on Hole Wizard . Select Straight Tap , Standard Ansi Metric, Type Bottoming Tapped Hole and Size M5 x 0.8. Under Options, select Cosmetic thread . Under End Condition, under "Thread", set Tap Thread Depth to 13 mm (The Hole Depth will be automatically changed to 15.40mm).

    

2.8) Select the Positions tab. Click on Right view. Select the face where the two holes are to be created. Left click to create the holes and use the Smart Dimension tool to position the holes as shown. Click on to finish.

2.9) For the hole M20 SQ. TH., because SolidWorks is unable to generate screw holes with square threads, we model the hole using an extruded cut. Click on the desired surface and sketch a Circle  of diameter 20mm as shown.

2.10) Click Extruded Cut  . Select Type Through all. Click on to finish.

 

FILLET THE JAW

2.11) Click on Fillet . Select the edge of the jaw as shown and set fillet radius to 13 mm.

 

2.12) You can now save this file by clicking File -> Save. Save this solid part as "Moving Jaw".
