Machine Vice

Step 1: Creating the Base

1.1) Click on New and select   to create a new part document

1.2) Select the Right plane (you have to select a plane before you can sketch). Click on Sketch . 

 

CREATING THE MAIN BODY

1.3) Using the Line and Smart Dimemsion tool, sketch the profile as shown with its dimensions.

1.4) Click on Extruded Boss/Base and type Depth 100 mm. Click on  .

1.5) Select the the top surface as shown

 

1.6) Click on Normal To . Sketch a Rectangle of length 149 mm (200-35-3-13=149) and breadth 38 mm as shown.

            

1.7) Click on Extruded Cut . Select type Through all. 

CREATING THE HANDLE AT THE SIDE

1.8) Click on Right view. Select the face and sketch a Rectangle as shown.

1.9) Click on Extruded Boss/Base and type Depth 38 mm. (20+18=38)

1.10) Click on Fillet . Select the edges of the handle as shown and set fillet radius to 18 mm. Click on  to finish.

   

 1.11) Select the top face of the handle. Click on Normal To .

    

1.12) Select Straight Slot . Click on the first and second points as shown to create the slot. Select any point as the third point and use Smart Dimension  to set the width to 28 mm.

 

1.13) Click on Extruded Boss/Base and type Depth 3mm. Click on . Select the top surface of the boss. Click on Sketch . Select Offset Entities. Set offset distance to 7 mm. Check the 'Reverse' direction box to ensure that the offset is inwards. Click on .

(Note: Ensure that you select Sketch  first before selecting Offset Entities .)
 

            

1.14) Click on Extruded Cut . Select Through All and click on to finish.

 

MIRRORING THE HANDLE

1.15) Select the face as shown. Click on Reference Geometry and select Plane. Set offset distance 50mm and select the 'Flip' box if necessary to ensure the plane lies in the middle of the model. Click on .

       

1.16) Click on Mirror  with the plane as the Mirror face/plane. Click Features to Mirror box and select the 4 main features of the handle to mirror: the extruded block, the extruded boss on top of the block, the cut-extrude and the fillet.

         

  

CREATING THE HOLES

1.17) Click on the Front  view and select the face. Sketch a Circle  with the dimensions shown. Click Extruded Boss/Base and type Depth 3mm.

       

1.18) Click on the face of the extruded boss. Sketch a concentric Circle   of diameter 20 mm. Click Extruded Cut . Select type Up to Next. Click on .

       

1.19) Click on the Back view and select the face. Click on Front view and sketch a circle of diameter 26 mm with its center coincident with the center of the hole created just now. Click on Extruded Boss/Base and type Depth 3mm. Click on .

1.20) Click on Back view, select the face of the extruded boss and sketch a concentric circle of diameter 13 mm. Click Extruded Cut . Select type Up to Next. Click on to finish.

 

1.21) We now need to create the tap holes (M5 x 0.8) using the Hole Wizard tool. 

Click on Hole Wizard . Under Type tab, select Straight Tap , Standard Ansi Metric, Type Bottoming Tapped Hole and Size M5x0.8. Under Options, select Cosmetic thread .Under End Condition and under "Thread", set Tap Thread Depth 13 mm. The Blind Hole Depth will automatically change to 15.40 mm.

Note: Select Cosmetic thread first before setting the "End Condition" to avoid the problem of "End Condition" reverting back to default value.

  

1.22) Select the Positions tab. Click on Back view. Select the face where the two holes are to be created. Left click to create the holes and use the Smart Dimension tool to position the holes as shown. Click on to finish.

   

 

FILLET THE MODEL

1.23) Click on Fillet and select the edge as shown. Set fillet radius to 13 mm. Click on to finish.

1.24) Click on Fillet and select the edges of the front and back bosses. Set fillet radii to 3 mm. Click on to finish.

1.25) Click on Fillet and select the 4 inner edges of the handles. Set fillet radii to 20 mm. Click on to finish.

1.26) Click on Fillet and select the 2 extruded boss edges as shown. Set fillet radii to 3 mm. Click on to finish.

1.27) You can now save this file by clicking File -> Save. Name this solid model as "Base".
