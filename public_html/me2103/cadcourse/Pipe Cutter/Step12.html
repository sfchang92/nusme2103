Pipe Cutter

Step 12: Bill of Materials Creation


12.1) Click on Open and select your assembled model. Click Browse, and open Pipe Cutter.SLDASM.

12.2) Click on Bill of Materials and select Indented with Detailed numbering.  Secondly, check the Display as one item number, then follow by Display all configurations of same part as one items. Click on    when done.

12.3) Following dialog, you may select Notes Area and click OK. Then, require to place a new list table  inside the graphical area as shown below.

Note: Double click the table, you may use the dialog as shown below to modify the table as table size and  font pattern. Similar to the software Microsoft Office.


12.4) You can now save this file by clicking File -> Save to update your assemblies drawing.
