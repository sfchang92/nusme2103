Pipe Cutter

Step 11: Exploded View Creation


11.1) Click on Open and select your assembled model. Click Browse, and open Pipe Cutter.SLDASM.

11.2) Click on Exploded View  and select components  for the explode step. Just  simply to drag an axis handle for a linear step. OR, drage and rotate the rings for rotational step.

You may hold Ctrl key to select and drag multiple components together.


11.3) You can now save this file by clicking File -> Save as Exploded_PipeCutter2.SLDASM.
